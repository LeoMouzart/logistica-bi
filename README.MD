# 📌 Sobre o Projeto

O **BI Logístico** é um dashboard interativo criado para monitorar operações logísticas e gerar insights estratégicos através de indicadores operacionais e financeiros.

O sistema permite analisar:

- 📦 Status das entregas
- 🚚 Performance de transportadoras
- 📈 Evolução de atrasos
- 💰 Indicadores financeiros
- ⭐ Satisfação do cliente
- 📊 KPIs operacionais
- 📤 Exportação de dados

# 🚀 Funcionalidades

## 📊 KPIs Operacionais

✔ Total de pedidos  
✔ Taxa de atraso  
✔ Valor extraviado  
✔ Satisfação média  
✔ Desvio médio de entrega  

---

## 🎛 Filtros Dinâmicos

O dashboard possui filtros interativos por:

- Transportadora
- Status da entrega
- Período

---

## 📈 Visualizações Analíticas

### 📅 Evolução Mensal
Análise temporal do volume de pedidos.

### 🚚 Performance das Transportadoras
Comparativo operacional entre transportadoras.

### 📦 Distribuição dos Status
Visualização dos status das entregas.

### 💰 Evolução Financeira
Análise financeira da operação logística.

### 📈 Correlação: Satisfação x Atraso
Análise do impacto dos atrasos na satisfação dos clientes.

---

## 🧠 Insights Automáticos

O sistema identifica automaticamente:

- Melhor transportadora
- Pior transportadora
- Indicadores operacionais críticos

---

## 📤 Exportação CSV

Exportação dos dados filtrados diretamente pelo dashboard.
